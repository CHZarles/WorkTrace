# v0.1.16（Windows）

- Planner UI 继续收敛信息密度，重点提升“打开后先看什么”的可读性：
  - 任务分区支持折叠，默认聚焦 `Overdue / Today`，其余分区默认收起。
  - 顶部增加 `Insights` 显隐开关，避免侧栏信息持续干扰主任务流。
- Calendar（周视图）可视范围改进：
  - 新增密度切换：`24h / Standard / Detail`。
  - `24h` 模式可完整查看全天时间轴，不再出现“看不全 24 小时”的问题。
- Report settings 分层重构：
  - 从单块密集表单改为 5 个分层入口：`Connection / Automation / Planner / Storage / Prompts`。
  - 每次仅展示一个设置层级，降低认知负担，减少误操作。
- 响应式细节优化：
  - Planner 头部在窄屏下自动改为分行操作区，按钮不再拥挤。

下载与更新：
- 安装版：`RecorderPhone-v0.1.16-windows-setup.exe`
