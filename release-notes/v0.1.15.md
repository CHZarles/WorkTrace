# v0.1.15（Windows）

- Planner UI 按“大厂产品”交互标准做了一轮可用性升级：
  - 顶部结构重排：统一入口（`New TODO`）、状态摘要与维度切换（`TODO / Calendar`）。
  - 信息架构升级：新增 `Open / Today / Scheduled / Reminder / Overdue` 概览卡片，减少扫读成本。
- TODO 维度升级：
  - 新增搜索、筛选（`All / Today / Scheduled / Reminder / Done`）与排序（`Smart / Due soon / Recently updated`）。
  - 列表分区展示：`Overdue / Today / Upcoming scheduled / Backlog`，`Done` 默认折叠，焦点更稳定。
  - 卡片信息更完整：时间、日期、提醒、逾期状态与快捷操作（编辑/完成/删除）。
- Calendar 维度升级：
  - 新增范围负载摘要（周内已排程 block 数与总时长 / 月内 open 数）。
  - 增加图例与交互说明（点击创建、拖拽改期、提醒/完成态）。
- 保持现有核心能力不变：
  - 周视图拖拽改期（15 分钟吸附）继续可用。
  - Reminder 与 TODO/Calendar 双维度保持一致。

下载与更新：
- 安装版：`WorkTrace-v0.1.15-windows-setup.exe`
