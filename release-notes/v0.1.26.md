# v0.1.26（Windows）

- Planner 继续简化：
  - 列表视图移除排序和筛选控件，保留搜索与核心分组。
  - 顶部区域收敛为模式切换、创建入口和简短状态统计。
  - Agenda 与部分辅助说明文案进一步压缩，减少视觉干扰。
- Calendar 保持周 / 月双视图：
  - 月视图继续保留。
  - 周视图抬高可视高度，并放宽单日列宽；窗口较窄时支持横向滚动，避免事件文字被压住。
- 更新检查链路修复：
  - 缺少 `build-info.json` 的已安装版本，也可以继续执行检查更新与安装更新。
  - 默认更新仓库回落到 `CHZarles/WorkTrace`，未手填仓库时仍可正常检查。
- Windows 打包与协议相关标识继续同步到 `WorkTrace`。
